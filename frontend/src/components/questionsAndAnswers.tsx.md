###  Overview

The `questionsComponent` is a React component responsible for rendering a form allowing users to select one user for each question. It also features a confirmation dialog to review the selected users before submitting the data. The component utilizes the `react-hook-form` library for form management and validation. 

###  Props

| Prop                | Type                       | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ------------------ | ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `sociogramId`       | `string`                    | ID of the sociogram associated with the form.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `negQuestions`      | `string[]`                  | Array of negative questions.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `posQuestions`      | `string[]`                  | Array of positive questions.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `users`             | `FormUsers[]`              | Array of users participating in the sociogram.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `mainUser`           | `FormUsers`                | The main user who is filling out the form.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |

###  State

| State               | Type                    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `isDialogOpen`        | `boolean`                | Flag indicating whether the confirmation dialog is open.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `isDisclaimerDialogOpen` | `boolean`                | Flag indicating whether the disclaimer dialog is open.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `finalPostData`      | `FormPostData`           | Object containing the final data to be submitted, including sociogram ID, main user information, and selected users for positive and negative questions.                                                                                                                                                                                                                                                                                                                                                                                 |
| `selectedUsers`     | `string[]`               | Array of selected user keys, each key representing a user ID and the question type (positive or negative).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `dialogPrompt`       | `string`                 | Text displayed as the description in the confirmation dialog, listing the selected users for each question.                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `translations`      | `{ [key: string]: string }` | Object containing translations for various UI elements.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

###  Context

| Context               | Type                     | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------- | ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `SubmitDataContext`  | `SubmitDataContext`       | Context used to share submit data between components.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `LanguageDataContext` | `LanguageDataContext`      | Context used to share language data between components.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

###  Form Schema

The `FormSchema` is defined using the `zod` library for validation. It specifies that the form must have a `items` array containing strings, each representing a selected user key. 

```typescript
const FormSchema = z.object({
  items: z.array(z.string()).refine(
    (value) => {
      // Code to check if exactly one user is selected for each question
      return true; // Return true if validation passes
    },
    {
      message: "You have to select exact 1 user for each question",
    }
  ),
});
```

###  Handlers

| Handler             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `handleCheckedChange` | Called when a checkbox for selecting a user for a question is checked or unchecked. It updates the `selectedUsers` array and the form field's value.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `handleConfirm`       | Called when the confirmation dialog's "Confirm" button is clicked. It closes the dialog, dispatches the `postFormData` action to submit the data to the server, and navigates to the "thank-you" page.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| `handleAccept`      | Called when the disclaimer dialog's "Accept" button is clicked. It closes the dialog.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `handleCancel`       | Called when the confirmation dialog's "Cancel" button is clicked. It closes the dialog.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

###  Hooks

| Hook                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `useAppDispatch`       | Hook used to access the Redux dispatch function. It allows the component to dispatch actions to update the Redux store.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `useNavigate`        | Hook used to access the `navigate` function from `react-router-dom`. It allows the component to navigate to different routes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `useForm`            | Hook used to manage and validate the form using `react-hook-form`. It provides methods to handle form submission, validation, and form field values.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `useContext`          | Hook used to access the `SubmitDataContext` and `LanguageDataContext`. It allows the component to read and update the context values.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `useEffect`          | Hook used to perform side effects, such as fetching data or setting up event listeners. It is used to fetch translations based on the language data from the context and to manually trigger validation on the form when the component mounts.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

###  Functions

| Function           | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `onSubmit`          | Called when the form is submitted. It validates the form data, prepares the final data to be submitted, opens the confirmation dialog, and updates the `dialogPrompt` with a list of the selected users.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `getTranslations`   | Function used to fetch translations based on the current language data.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
